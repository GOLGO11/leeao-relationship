《李敖报刊集》第五轮校对记录

本轮范围：
专项处理“出版管制、查禁、办刊办报行政线”。重点查看《被封杀的〈千秋评论〉》《万岁评论》缘起、《乌鸦评论》休刊告白、《求是报》创刊说明、《求是报》相关篇章中的新闻局、警总、出版法、执照、登记、查禁、抢书、封杀等线索。

主要处理：
1. 新增独立分类“出版管制”。
   这个分类用于区分“掌握出版、报刊、电影、宣传审查或行政许可权力的人”，避免他们和实际办刊、编辑、出版协作者混在“出版”里。

2. 调整管制线人物：
   邵玉铭：改为“出版管制 / 媒体 / 公职人员 / 政治人物 / 公共论战”。本书中他以新闻局局长身份核发《求是报》执照，是出版行政关卡人物。
   熊钝生：改为“出版管制 / 公职人员 / 出版 / 公共论战”。本书中他是内政部掌管出版事业、明告不准登记杂志的行政人物。
   谢然之：改为“出版管制 / 政治人物 / 党务宣传 / 公职人员 / 公共论战”。本书中他作为“防李敖之口”的官方党务宣传人物出现。
   楚崧秋：改为“出版管制 / 党务宣传 / 媒体 / 公共论战”。本书中他与胡茵梦被中影冷冻、文工会警告线有关。

3. 保持协作者不误归：
   林秉钦、叶圣康、苏荣泉、施复华仍保留在“出版 / 编辑 / 创作资料协助 / 媒体”等协作线，不归入“出版管制”。
   高信疆仍以“媒体 / 出版 / 通信 / 朋友 / 证人见证 / 见面 / 公共论战”为主，他是受压力暂停李敖文字的媒体中介，不是管制权力人物。

4. 补充短语证据：
   新闻局局长邵玉铭、熊钝生处长、高信疆受到压力、楚崧秋请我吃饭、施复华先生主持业务、苏荣泉先生等。

5. 新增人物关系：
   谢然之—楚崧秋：文工会系统 / 言论与演艺管制。
   熊钝生—邵玉铭：出版行政 / 登记与报纸执照。

校对经验：
“出版管制”不能等同于“出版”。一边是管制者，例如新闻局、内政部出版行政、党务宣传系统；另一边是实际协作者，例如预编、付印、出版公司、订户善后、海外发行业务。后续遇到报刊类文本，应先问：这个人是在帮李敖出版，还是在限制李敖出版？
