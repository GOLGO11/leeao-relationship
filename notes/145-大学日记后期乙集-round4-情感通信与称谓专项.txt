《大学日记后期乙集》第四轮：情感、通信与称谓待考专项

本轮目标：
复查本书中情感线、通信线和称谓待考人物，重点避免两类错误：一是现实往来人物被误判为神交；二是只有称谓但无法确认全名的人被强行合并。

处理结果：
1. 补入漏收人物：
   - Bonnie：本书出现 18 次，涉及见面、索糖、写信、送诗、谢师宴谈话等，列为“情感 / 见面 / 通信”。
   - Stone：1959 年 2 月“史一的 Stone”，列为“情感 / 见面 / 称谓待考”。
   - 程有美：涉及写信、赠物、写诗催毕业礼物，列为“朋友 / 通信 / 见面”。
   - 王大姐：1959 年 7 月“夜请施珂及王大姐于重庆”，列为“见面 / 称谓待考”。
   - 曲先生：1959 年 7 月“与曲先生小谈”，并在甲集也有线索，列为“见面 / 称谓待考”。

2. 修正误分身份：
   - 袁兆真：由“神交引用”改为“情感 / 见面 / 通信”。依据包括题赠照片、打油诗、约谈等现实往来。
   - Rosa：去掉“称谓待考”，改为“情感 / 通信 / 见面”。本书有校园见面，其他书有通信与单恋线。
   - 咪咪：去掉“称谓待考”，改为“情感 / 同学 / 见面 / 通信”；并加入搜索别名“涂咪咪”。
   - 黄淑兰：由“通信 / 情感”扩充为“通信 / 情感 / 见面 / 朋友”。
   - 小玄：保留“称谓待考”，但补入“见面”，即“情感 / 见面 / 称谓待考”。

3. 保留待考：
   - 李教官：证据明确为现实见面与军训/教官身份，但无法确认全名，保留“称谓待考”。
   - 刘老板：本书中是关颂韬同班者，跨书又会指刘长乐语境；暂不强行合并，保留“称谓待考”。

本轮经验：
1. 英文名不会被中文姓名规则稳定捕捉，日记类书里的英文名应单独扫描，例如 Bonnie、Stone。
2. “称谓待考”不能一律清掉；能从后文确认身份的才去掉，不能确认全名的应保留，避免伪精确。
3. 情感人物往往同时有“见面 / 通信 / 赠物 / 诗文题赠”等多重边，不能只落在“情感”一个节点。
4. 同一个称谓可能跨书指向不同人，例如“刘老板”，需要按证据分层，不急于合并。

下一轮建议：
第五轮建议做“英文名、单字简称与小称呼专项”。重点查 Bonnie 周边的“黄、王、程”、日记中的 Y、L、小羊、小丢、小梅等，判断哪些能还原成人物，哪些应作为不可确认称谓保留或排除。
