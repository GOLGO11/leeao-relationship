《上下古今谈》第五轮：本书直接关系与出版停刊线专项校对

处理时间：2026-05-07

本轮目标：
1. 校对本书中与李敖现实接触较近的人物线索。
2. 重点检查《台湾日报》《中华日报》《文星》相关的出版、办报、专栏、停摆、通信和公共论战关系。
3. 合并同一人物的笔名、简称，避免重复节点。

本轮结果：
- 全局导出：19 本，1195 章，2348 个人物节点。
- 《上下古今谈》：61 章，196 个人物节点。
- 重复姓名检查：0。
- 别名残留检查：吴心柳、晓华、孟能、黄主席均无独立残留节点。
- CSV/TXT/网页数据已随脚本全量重跑。

本轮归并：
- 吴心柳 -> 张继高。原文为“张继高（吴心柳）”，不应拆成两人。
- 晓华 -> 夏晓华。用于“晓华兄”等书信称呼。
- 重则兄 -> 重则。
- 孟能继续归并到萧孟能，未形成“孟能”独立节点。

本轮补入：
- 刘一成：《中华日报》记者，写《婚谣困扰刘秀嫚》相关专栏，归入媒体 / 公共论战。
- 洪敬思：《中华日报》报道《从绚丽归平淡——刘秀嫚弃虚荣》的作者，归入媒体 / 公共论战。

本轮纠偏：
- 刘秀嫚：去掉“情感 / 通信”，保留媒体 / 出版 / 公共论战。理由是原文明确把“闹恋爱”写成报纸捕风捉影和谣言，不应把谣言转成真实情感关系。
- 洪敬思：从“情感”纠正为媒体 / 公共论战。
- 何秀煌：从姻亲 / 证人见证纠正为朋友 / 学术 / 出版 / 公共论战。本书语境是“青年朋友”和《文星丛刊》作者。
- 张继高：保留媒体 / 出版 / 朋友 / 见面 / 公共论战；其“预言”与停刊压力线直接相关。
- 张明炜：保留媒体 / 出版 / 朋友 / 公共论战；其建议“上古多谈，下今少谈”属于本书发表压力线。
- 夏晓华：保留出版 / 媒体 / 通信 / 见面 / 朋友 / 公共论战；本书中有请李敖写专栏、请吃饭、来信、承受外界压力、停摆处理等多重直接关系。

复核说明：
- 萧孟能在本书中通过“孟能坚持最好九月一号宣布，二十五号出书”出现，属于出版线，但因其全局出现极高，逐书索引的高频截取列表未显示；人物主数据、证据、CSV/TXT 总表中保留。
- 刘一成、洪敬思为低频媒体人物，也可能不出现在逐书索引高频列表中，但人物主数据与导出总表完整保留。

本轮校对经验：
1. 笔名括注必须优先归并，如“张继高（吴心柳）”。
2. 书信称呼中的简称要回归全名，如“晓华兄”归夏晓华。
3. 谣言、报章捕风捉影、李敖自嘲式文字，不应直接转成“情感”关系。
4. 报纸记者、附录作者、报道作者应纳入媒体 / 公共论战，而不是按其报道内容误归情感。
5. 直接关系线要区分：夏晓华是现实出版合作与通信见面；刘秀嫚主要是媒体/专栏/公共论战对象；张继高、张明炜是出版压力线中的朋友与媒体人物。
