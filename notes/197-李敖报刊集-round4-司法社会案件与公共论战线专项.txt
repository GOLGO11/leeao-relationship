《李敖报刊集》第四轮校对记录

本轮范围：
按第三轮建议，专项处理本书中的司法、社会案件与公共论战线。重点仍在《李敖与“编辑室报告”》，并兼看《文星杂志选集》导言、《万岁评论》缘起、《求是报》相关篇章中的出版管制、查禁、报禁线索。

主要处理：
1. 调整“监察委员弹劾法官”线：
   陶百川从“司法人员”主线改为“公职人员 / 政治人物 / 公共论战 / 历史类比引用”。
   黄宝实从单纯“诗文出处 / 出版 / 神交引用”改为“公职人员 / 诗文出处 / 出版 / 公共论战 / 历史类比引用”。
   高友唐从“司法人员”倾向中移出，改为“公职人员 / 诉讼 / 历史类比引用 / 公共论战”。
   这样可以避免把“监察委员、法政评论者”误读成被批判的法官。

2. 调整法学文章作者线：
   李声庭从“律师代理”主线改为“学术 / 诗文出处 / 人权救援 / 公共论战”。本书里他主要作为法学教授、文星文章作者出现，不是李敖现实官司代理律师。

3. 补入案件节点：
   新增张子岐，分类为“政治人物 / 社会案件 / 诉讼 / 公共论战”，对应“张子岐一案”与议员言论不负责权的讨论。
   张韵淑增加“火场双尸案”短语证据，保持“社会案件 / 被害人 / 诉讼”。

4. 新增人物关系：
   陶百川—黄宝实：监察委员 / 共同弹劾法官。
   李声庭—张韵淑：冤狱评论 / 火场双尸案。
   陆啸钊—张韵淑：慎刑评论 / 火场双尸案。

校对经验：
本轮再次确认，法政类文本要分清四种角色：
1. 法官、检察官、律师等司法职业人物。
2. 监察委员、部长、新闻局等公职或行政权力人物。
3. 案件当事人、被害人、被告等社会案件人物。
4. 法学文章作者、评论者、历史参照人物。
这些角色常在同一段里互相缠绕，但分类上不能偷懒合并。

本轮结果：
全量重建后：4714 人物，52 本书，3313 章。
《李敖报刊集》当前：18 章，136 人物。
CSV、TXT、JSON、网页数据已同步重建。

下一轮建议：
进入《李敖报刊集》第五轮，做“出版管制、查禁、办刊办报行政线专项”。重点检查邵玉铭、熊钝生、谢然之、楚崧秋、高信疆、林秉钦、叶圣康、苏荣泉、施复华，以及新闻局、警总、出版法相关段落中能归到具体人物的线索。
