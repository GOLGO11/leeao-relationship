《千秋万岁乌鸦求是合集》第三轮：政论人物与学术思想线校对

一、处理范围

本轮重点检查本书高频的思想、学术、政论人物，尤其区分：
现实交往、师生关系、公共论战、出版关系、诉讼关系、神交引用。

重点人物：
殷海光、胡秋原、徐复观、胡适、孔子、朱熹、陈鼓应、张建邦、居浩然、林正杰、康宁祥、孙中山、蒋介石、蒋经国、伽利略。

二、本轮改正

1. 陈鼓应

原分类：
师生 / 学术 / 同学同事 / 公共论战

现分类：
学术 / 公共论战

理由：
原文里的“学生”主要指陈鼓应是殷海光的学生，不是李敖与陈鼓应之间的师生关系。此前被线索词误拉成“师生”，容易误导。陈鼓应在本项目中更适合作为学术人物与公共论战人物。

2. 徐复观

原分类：
学术 / 出版 / 诉讼 / 公共论战 / 神交引用

现分类：
学术 / 出版 / 诉讼 / 公共论战

理由：
徐复观不是纯粹古人或远距思想引用人物。他与李敖有现实的论战、出版、诉讼线索，因此不宜列入“神交引用”。保留现实关系线更清楚。

3. 胡秋原

原分类：
公共论战 / 出版 / 学术

现分类：
公共论战 / 出版 / 学术 / 诉讼

理由：
胡秋原与李敖长期论战，并有互告、法院判决、赔款等诉讼关系。只列公共论战会漏掉重要现实关系。

三、保留判断

殷海光
保留：师生 / 出版 / 学术 / 公共论战。
理由：殷海光是李敖现实老师，也是思想、出版与文化论战核心人物。

胡适
保留：通信 / 见面 / 学术 / 学术行政 / 公共论战。
理由：胡适在全局中不仅是思想资源，也有通信、见面或准现实交往线索，因此不降为纯神交。

孔子
保留：神交引用。
理由：属于思想、典籍和传统批判中的引用对象，不是现实人物关系。

朱熹
保留：神交引用 / 学术。
理由：作为宋代理学人物被讨论，主要是思想引用；“学术”仅表示思想学术对象，不表示现实交往。

伽利略
保留：神交引用。
理由：本书中以科学史、异端与平反范例出现，属于典型神交/引用人物。

孙中山
保留：政治人物 / 公共论战 / 神交引用。
理由：孙中山既是现代政治谱系中的核心人物，也主要以思想、遗教、国共论述对象出现，因此保留神交引用。

蒋介石、蒋经国
保留：政治人物 / 公共论战。
理由：二人虽有大量历史与政治批判文本，但在李敖作品中也构成现实政治压迫与论战背景，不简单归为神交。

张建邦
保留：政治人物 / 财产债务 / 公共论战。
理由：本书中以丑闻、家族财务、政治职务和公共批评线为主。

居浩然
保留：通信 / 朋友 / 学术 / 出版 / 见面。
理由：居浩然和李敖有现实通信、朋友、学术出版关系，不是纯思想引用。

林正杰
保留：政治人物 / 媒体 / 公共论战。
理由：主要处于党外、民进党、杂志和政治批评线中。

康宁祥
保留：政治人物 / 公共论战。
理由：主要是政治人物与公共论战对象。

五、校对经验

1. “学生”“老师”线索必须判断是谁和谁的关系。出现“殷海光的学生陈鼓应”时，不能自动推为李敖与陈鼓应的师生关系。

2. 神交引用不等于所有思想人物。若人物与李敖有现实诉讼、出版、论战、通信、见面关系，应优先保留现实关系，不要因为学术文本很多就加入神交。

3. 古典人物、科学史人物、宗教思想人物可以保留神交引用，尤其是孔子、朱熹、伽利略这类作为思想范例、批判对象、历史类比出现的人。

4. 现代政治人物要看是否构成现实政治背景。蒋介石、蒋经国、孙中山不是普通思想引用，他们在李敖论述中还承担党国、压迫、制度、历史责任等公共论战轴线。
