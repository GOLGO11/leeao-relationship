《李敖语录》第六轮：亲友师生与直接关系检查

本轮目标：
检查《李敖语录》中“见面、通信、师生、朋友、出版、公共论战”等直接关系类，区分真实直接关系、旧文复用关系和语录体邻词误触发。

本轮结果：
全库生成后为 39 本、2494 章、3803 人、8967 条身份边、119 条人物关系边。
《李敖语录》目前为 31 章、582 人。

本轮校对动作：
1. 钱穆保留“学术 / 通信 / 公共论战 / 历史类比引用”，去掉缺乏直接证据的“见面”。其通信证据来自李敖高中时期写信请教、钱穆回信。
2. 宋希濂保留“军事人物 / 通信 / 出版 / 公共论战”，去掉“见面”。其直接关系重在书信、资料寄送和《鹰犬将军》相关出版。
3. 陈映真去掉误触发的“见面”，调整为“出版 / 编辑 / 政治异议者 / 同案狱友 / 公共论战”。语境里是李敖对陈映真及《人间》的评论，不是李敖见陈映真。
4. 汝清主类从“出版”改为“情感”，保留“见面 / 通信 / 出版”。
5. 张继高补入“通信”，保留“媒体 / 出版 / 朋友 / 见面 / 公共论战”。
6. 马尔腾从“通信”改为“诗文出处 / 历史类比引用 / 神交引用”，因为他是格言式引用人物，不是给李敖通信的人。
7. 杨宝琳从“待复核”改为“政治人物 / 公共论战 / 神交引用”。
8. 清理“高傲”“暴政斗”两个抽象词或句子残片。

复核保留：
李方桂、林青霞、何凡、叶公超、台静农等人的“见面”有跨书直接证据，保留。
高信疆、张继高等人的“通信 / 朋友 / 出版”有书信、专栏、出版合作证据，保留。

校验：

经验：
“见”“信”“朋友”在语录体里不一定指李敖与该人物的直接关系，常常只是“谈某人时出现的邻词”。直接关系类必须看完整片段：谁见谁、谁写信给谁、谁是朋友。若只是引用某人一句话，应归“诗文出处 / 神交引用”；若是评论某人的文章或杂志，应偏向“出版 / 公共论战”。

下一轮建议：
