《虽千万人，李敖往矣》第六轮：高频政治人物与残余分类收束检查

本轮目标
检查本书高频政治人物与政治语境残片，重点防止“主席、父母、学生、法院、见面”等邻词把人物错误拉进亲属、师生、监狱行政或强直接关系。

处理结果
1. 李庆华：移除“亲属”。文本中的母亲、父亲线索是李庆华家庭背景，不是李敖亲属关系；保留“政治人物 / 朋友 / 见面 / 公共论战”。
2. 齐白石：移除“政治人物 / 师生 / 见面”。本书与全局证据主要是画家、艺术典故和再传弟子式引用，改为“文艺音乐 / 历史类比引用 / 神交引用”。
3. 许水德：移除“监狱行政”。“法院是国民党的”等语境不是监狱行政；全局保留政治人物、公共论战，并因其他书有偶遇证据保留“见面”。
4. 辜振甫：移除“见面”。现有强证据主要是荣星花园、辜家财产与公共争议，不宜让“华严到我家谈判”误读成辜振甫本人见面；改为“政治人物 / 财产债务 / 公共论战”。
5. 清理残片：“全会议”“钱复回”“荣誉主”加入停用词。它们分别来自“国家安全会议”“钱复回忆录”“荣誉主席”的断词，不应进入人物图。

本轮复核结论
蒋介石、毛泽东、马英九、李登辉、陈水扁、连战、孙中山等本书高频政治人物分类保持稳定，没有被亲属、媒体、虚拟人物或见面类关系污染。慈禧、袁世凯、林肯等带“小说剧本人物”的多重身份来自此前小说类书目，不是本书误判。

校对经验
1. 高频政治人物不能只看身份本身，还要看强关系是否被邻词误触发；“亲属、见面、师生、监狱行政”比“政治人物、公共论战”更需要证据。
2. “某人的父母/妻子/老师/学生”不能反推成李敖关系，除非证据明确指向李敖。
3. “荣誉主席、国家安全会议、某某回忆录”这类固定短语容易切出假人名；政治文本里要把职称残片单列为停用词。
4. 对古今名人，若只是类比、典故、引用，优先归入“历史类比引用 / 神交引用”，不要因为附近出现“学生”“见”就升级为直接关系。
