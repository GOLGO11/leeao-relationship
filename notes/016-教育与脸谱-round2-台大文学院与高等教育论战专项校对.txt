《教育与脸谱》第二轮：台大文学院与高等教育论战专项校对

本轮目标
围绕台大文学院、高等教育怪现状讨论、考试法批评等章节，补齐论战名单中的真实人物，并修正“学者、学术行政、文章作者、考试评分人”之间容易混淆的身份边。

本轮处理
1. 补入或校正台大文学院、中文系、哲学系相关人物：
   陈康、王叔岷、劳榦、洪炎秋、朱光潜、毛子水、台静农、沈刚伯、王洪钧、钱思亮等。

2. 补入高等教育讨论中的署名作者和回应者：
   李朴生、谢冰莹、乔龄、何细雨、潘东坡、楚卿。

3. 补入考试法批评相关的阅卷、考试、政治行政人物：
   沈兼士、马国琳、陈玉科、陆锡光、王立哉、张默君、莫德惠。

4. 修正身份边：
   洪炎秋：从“学术 / 师生 / 公共论战”修正为“学术 / 出版 / 公共论战”。
   朱光潜：保留“神交引用”，同时补“公共论战”，因为本书里不只是一般引用，也进入具体教育批评语境。
   王叔岷、劳榦、陈康：统一按“学术 / 公共论战”处理，避免误挂为李敖个人师生关系。
   张默君：保留既有“政治人物 / 见面”，并补入本书语境下的“公共论战”。
   陈彦增：在既有朋友、通信、见面、见证关系外，补入本书中的文章署名与讨论关系，“出版 / 公共论战”。

5. 暂不收入“王道”：
   文中“有了一个王道”像是社会案件中的人名，但“王道”本身短且常作普通词，误伤风险高。本轮不做全局节点，留到后续“考试院 / 考试法 / 社会案件”专项反查。

校对经验
1. 惊世杂文类里，“文章标题中的人名”不一定代表李敖直接关系，但只要是李敖正在公开评论、回应、批判或借其文章展开论述的人，应进入“公共论战”。
2. “教授、先生、系主任、校长、院长”要拆开判断：普通教授多为“学术”，实际掌权或被批评行政责任者才加“学术行政”。
3. “论文、评论、读某文、讨论文章”的署名人，优先补“出版 / 公共论战”；如果其本职明确为学者，再补“学术”。
4. “考试评分人、典试委员、院长、部长”不自动归入司法或情治；本书语境更常是“公共论战 / 学术 / 政治人物”。
5. 短词疑似人名需要更严格，尤其“王道”这种高度普通词，不能为了求全而放大误收。
