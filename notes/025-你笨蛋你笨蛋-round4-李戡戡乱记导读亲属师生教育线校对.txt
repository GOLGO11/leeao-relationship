新增或补正人物

   理由：本章后篇核心人物，是李敖之子，也是《李戡戡乱记》的作者与教育批判线索中心。

2. 李谌：亲属。
   理由：作为李戡妹妹、李敖女儿出现，低频但关系明确。

3. 张松涵、张仁龙、张仁园、张仁宁。
   理由：张松涵是李敖父亲老友，张家三兄弟与初到台中生活记忆有关；此前漏收。

4. 王福霖：师生。
   理由：童军老师，直接出现在李敖初二求学经历中。

5. 严方、严正、严谅、严倬云。
   理由：严侨被捕后家属线明确；严方、严正、严谅是严侨三个孩子，严倬云是严侨妹妹、辜振甫妻子。

6. 牟琴、杨肇南、杨肖震。
   理由：台中一中老师，并在白色恐怖语境中被捕，补为师生 / 政治异议者。

身份边校正
1. 严侨：师生 / 学术 / 朋友 / 见面 / 政治异议者。
   理由：他既是李敖老师，又是深交与思想影响者，并在白色恐怖中被捕。

2. 林蒨：师生 / 见面。
   理由：她以“严师母”出现，不宜继续落成姻亲；关系来源是严侨师生线和实际照应往来。

3. 王墨林：政治人物 / 同学同事 / 朋友 / 见面。
   理由：他是李敖父亲北大同班同学、立法委员，并帮助李鼎彝在台中一中任教。

4. 张立纲：学术 / 同学同事。
   理由：演讲比赛同场人物，后来成为院士级学者。

5. 张立豫：姻亲 / 学术。
   理由：张立纲之兄、李敖四姐夫，同时文本明确提及学者线。

6. 李远哲：科学家 / 学术 / 学术行政 / 政治人物 / 公共论战。
   理由：本章以中央研究院院长、教改负责人与立法院质询对象出现，不能只保留科学家或政治边。

7. 龙应台：媒体 / 学术 / 公共论战。
   理由：本章在文化局长、《大江大海》与台湾叙事批判中出现，不应作为神交引用。

误切清理
1. “都说”来自“总统都说”，不是人物。
2. “余痕”来自“殖民余痕”，不是人物。
3. “严正”是高风险普通词，已加上下文限制，只在“儿子严正尚小”等严侨家属语境中保留。

本轮校对经验
1. 书名中的叠字可能掩盖本名，例如“李戡戡乱记”应回收为李戡，不能只停用“李戡戡”。
2. 亲属小人物低频但重要，尤其会支撑关系图里的家庭支线；但常见词如“严正”必须加上下文护栏。
3. “师母”不是姻亲，后续遇到“某师母”应优先落在师生关系延伸或见面照应线。
4. 教改与立法院质询线里，同一人物可能兼具学术行政、政治人物、公共论战身份，例如李远哲。
5. 白色恐怖段落里的老师被捕，不能只归师生；若文本明确说共产党、匪谍、被捕、折磨，应加政治异议者边。
