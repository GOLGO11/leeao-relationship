《大学札记》第一轮：首轮抽取与题名残片清理

本轮范围：
006.沉思日记类 / 001.大学札记

文本情况：
1. 本书正文 171 篇，加《大学札记》前记，共 172 个内容章节进入抽取。
2. 日记体中混有三类人物：
   - 台大时期现实往来人物，如张世民、殷海光、孟大中、翁松燃、陈彦增、姚从吾、刘博崑、沈鸿骏等。
   - 思想与文学引用人物，如胡适、罗素、曾国藩、梁启超、斯宾诺沙、歌德、莎士比亚、屠格涅夫等。
   - 宗教、虚拟或修辞对象，如上帝。

首轮结果：
1. 加入本书后，当前累计 42 本书、2730 章。
2. 《大学札记》当前索引：172 章、114 人。
3. 全局当前：3858 人、9126 条身份边。

本轮清理：
清理以下题名、句子、引文或贪婪切词残片：
石心肠、严肃、温暖、解自己、和血吞、单调生、谢公曰、松自己、余味、从孤独、牛津运、能石块、安详、余独好、金言、廉詹姆、史系迎、屠格涅、鲁达克、富尔、通信。

本轮补齐：
1. 刘博崑：明确“早上与刘博崑谈七十分钟”，归入“见面/朋友”。
2. 沈鸿骏：明确“与沈鸿骏出外散步半小时”，归入“见面/朋友”。
3. 陈淑平：明确“夜赴陈淑平家开历史系迎新会筹备会”，归入“见面/同学”。
4. 加富尔：不是常规中文姓名正则可抓的人名，按短语补入“政治人物/历史类比引用/神交引用”。

本轮保留：
1. 大量思想、文学、历史人物暂时保留，因为《大学札记》本就是自我修养札记，引用人物密度高。
2. 上帝保留为虚拟/宗教/神交类人物。
3. 张世民、孟大中、翁松燃等现实往来人物保留，后续需做直接关系专项。

校对经验：
1. 沉思日记类的题名常常是抽象词，容易被误切成人名。
2. 这类书不能简单压低人物数，因为引用人物本来构成李敖思想关系的一部分。
3. 非常规译名或历史人物，如“加富尔”，需要短语补入，而不是靠通用中文姓名正则。
4. “通信”这类词在多书里都可能被误当人物，本轮已作为全局残片清理。

校验：
1. python -m py_compile scripts/extract_relationships.py 通过。
2. node --check data/relationship-data.js 通过。
3. node --check assets/app.js 通过。
4. CSV 表头保持为：姓名,分类,主分类,可信度,出现次数,相关出现,涉及书数,涉及章节数,主要书目,线索。
5. scripts/__pycache__ 已清理。

下一轮建议：
做第二轮“现实往来人物专项”。重点核对张世民、孟大中、翁松燃、陈彦增、殷海光、台静农、方豪、姚从吾、吴相湘、刘博崑、沈鸿骏、陈淑平、史静波等，区分见面、通信、同学、朋友、师生、学术关系，避免被思想引用人物混淆。
