《你笨蛋，你笨蛋》round5：出版文友与神交引用线校对

本轮目标
专项校对〈送高信疆归大陆序〉、〈准伤痕文学〉、〈我写《上山·上山·爱》〉、〈给华严的一封信〉、〈《大江大海骗了你》书衣词〉等章节，重点处理作家、出版人、媒体人、题签人、追思见证者与纯文本引用人物之间的边界。

   姓名,分类,主分类,可信度,出现次数,相关出现,涉及书数,涉及章节数,主要书目,线索

新增或补正人物
1. 张大春：出版 / 媒体 / 公共论战。
   理由：在高信疆追思会上当众讲出李敖替高信疆出钱出力之事，是出版文友线中的见证人物。

2. 张隆延：文艺音乐 / 出版。
   理由：在华严写真集题签疑误中出现，虽低频，但是真实题签/书艺线索。

3. 文心：媒体 / 称谓待考。
   理由：文本有“文心对谈”“文心她们”，暂保留为节目/媒体相关称谓待考节点，后续若查明全名再归并。

4. 余老板 -> 余纪忠。
   理由：高信疆“为了我开罪余老板”处，结合前文出版媒体语境，应归并到余纪忠，不另立模糊节点。

身份边校正
1. 朱俊晓：政治人物 / 公共论战。
   理由：本书中围绕立委选举、立法院相见、政党提名劝说出现，主分类不应停在待复核。

2. 萨特：神交引用 / 学术。
   理由：在〈准伤痕文学〉中作为法国文学家与诺贝尔文学奖典故被引用，无现实接触。

3. 韩愈：神交引用 / 学术 / 政治人物。
   理由：在〈送高信疆归大陆序〉中借《送李愿归盘谷序》作古文参照，应优先作为神交引用。

4. 龙应台：媒体 / 学术 / 公共论战。
   理由：〈《大江大海骗了你》书衣词〉集中批判其文章与历史叙事，不是一般神交引用。

5. 高信疆：媒体 / 出版 / 通信 / 朋友 / 证人见证 / 见面 / 公共论战。
   理由：本轮确认其在追思、出版文友、朋友救助与见证线中均有明确角色。

误切清理
1. “陆序”来自“归大陆序”，不是人物。
2. “余老板”不保留独立节点，已并入余纪忠。

本轮校对经验
1. 文章题名容易切出假人名，例如“归大陆序”误出“陆序”，标题残片要优先清理。
2. 追思会、题签、写真集、书衣词这类出版外围文本会出现低频真实人物，不能只按次数删。
3. 历史文学典故人物如韩愈、萨特，若只是文本参照，应让“神交引用”在前，避免被误读为现实交往。
4. “老板”“大使”“先生”等称谓要谨慎：能可靠归并就归并，不能归并才作为称谓待考。
5. 书衣词中的被批判对象是公共论战人物，不等同于神交引用。
