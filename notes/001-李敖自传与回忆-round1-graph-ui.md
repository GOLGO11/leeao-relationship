# 《李敖自传与回忆》图谱 UI 与身份边调整

生成时间：2026-05-04

## 本轮修改

1. 页面主视图从表格改为人物关系网络拓扑图。
2. 图谱采用三层结构：
   - 中心节点：李敖
   - 身份分类节点：直接接触/往来、家人/情感、出版/学术/媒体、司法/诉讼/审判、政治/公共论战、神交/思想引用、间接提及/待复核
   - 人物节点：本书抽取的人物
3. 每个“人物-身份分类”都是独立关系边，存放在 `identityLinks` 中。
4. 人物节点保留出现次数 `occurrences`，身份边保留该分类证据次数 `count`。
5. 点击人物节点可查看出现次数、身份边、线索和证据片段。
6. 点击身份分类节点可查看该分类下的相关人物。

## 当前数据

- 人物节点：155
- 身份边：198
- 书目：1
- 章节：16

## 重要说明

当前不再把多重身份合并成一个最终人物身份判断，而是先保留为独立分类关系，后续再由人工逐个合并、删除或细分。

例如 `胡适` 当前有多条身份边：

- 胡适 -> 神交/思想引用
- 胡适 -> 直接接触/往来
- 胡适 -> 家人/情感

这些边暂时并列保留，不在本轮做最终裁决。

## 下一步建议

下一轮建议先处理“身份分类体系细化”，把粗分类拆成更可用的关系类型，例如：

- 直接接触/往来：见面、通信、拜访、师生、同事、朋友
- 家人/情感：亲属、姻亲、恋爱、婚姻、称谓待考
- 出版/学术/媒体：作者、编辑、出版人、学术影响、论敌
- 司法/诉讼/审判：法官、检察官、律师、被告、同案、狱友
- 政治/公共论战：政治人物、情治人员、党派人物、公共评论对象
- 神交/思想引用：古人、外国思想家、文学人物、宗教人物

细分类确定后，再回到这 155 个人物逐个校对身份边。
