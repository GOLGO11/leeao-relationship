《李敖报刊集》第八轮：closeout 前全书体检

本轮目标：
1. 按低置信名单回到原文复核，重点查历史法政人物、文学引用人物、封面人物和报刊评论者是否串类。
2. 检查是否仍有残片名、重复名或称谓名独立成节点。
3. 为本书 closeout 做最后一轮数据健康记录。

本轮校正：
1. “海伦凯勒”从“海伦”中拆出。
   - 原文是《编辑室报告》封面人物“海伦凯勒”，此前被“海伦”节点吸收，容易与希腊神话人物海伦混淆。
   - 现新增“海伦凯勒”节点，分类为：诗文出处 / 人权救援 / 历史类比引用 / 神交引用。
   - “海伦”保留为虚拟人物/神话人物，不再承接本书海伦凯勒证据。

2. “沈家本”分类修正。
   - 原文谈陆啸钊《晚清时代法律思想的大论战》，沈家本是晚清修律与法律现代化的历史法政引用。
   - 从单纯“公共论战”改为：司法人员 / 诗文出处 / 历史类比引用 / 神交引用。

3. “吉卜林/吉卜龄”别名与分类修正。
   - 原文作英国诗人吉卜林，本项目沿用既有规范名“吉卜龄”，并将“吉卜林”作为别名。
   - 分类从出版链修正为：诗文出处 / 历史类比引用 / 神交引用。

4. “叶慈”分类修正。
   - 本书是与吉卜龄同段出现的诗人引用，不应被归为出版链。
   - 分类修正为：诗文出处 / 历史类比引用 / 神交引用。

5. “凯末尔”分类补强。
   - 原文说“凯末尔的精神——尤其是分权精神”，属于政治/军事改革人物的历史引用。
   - 分类补为：政治人物 / 军事人物 / 历史类比引用 / 神交引用。

6. “蒋芸”分类修正。
   - 原文是蒋芸女士批评李敖的文字，属现实报刊评论者。
   - 分类从神交倾向修正为：诗文出处 / 出版 / 公共论战。

体检结果：
1. 单书临时抽取通过：137 人、1 本、18 章。
2. 全量重建通过：4716 人、52 本、3313 章。
3. 《李敖报刊集》维持 18 章、137 人。
4. 未发现“陈先生”独立节点；“陈先生说/陈先生文章”等仍归并到陈嘉宗。
5. 未发现“吉卜林”重复节点；已作为“吉卜龄”别名进入搜索/导出。
6. “包龙图”在本书仍保留为虚拟人物，符合“包龙图探阴山”文学/戏曲典故语境。

本轮经验：
1. 人名包含关系要特别小心：海伦凯勒不能被短名“海伦”吞掉。
2. 报刊集中的“先生/女士 + 写作/批评/后记”多半是现实作者或评论者，不应轻易打成神交。
3. 历史法政人物如果是被文章引用，应保留其本职身份，同时加历史类比/神交引用；不宜只按“论战”词面归类。
4. 诗人、小说家、思想家出现在引文和封面短评中，优先视为诗文出处/历史引用，再判断是否有现实接触。
