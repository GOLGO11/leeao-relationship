《李语录》第三轮：政治人物与历史典故专项

本轮目标
校对《李语录》中政治人物、历史人物、学术典故和文学引用的身份边，避免把真实人物、小说剧本人物、神交引用混成同一类关系。

本轮修正
1. 肯尼迪
   - 修正为：政治人物 / 剧情人物 / 文学人物 / 历史典故 / 神交引用
   - 原因：他在《阳痿美国》中有小说剧本身份，但在全局人物图里不能让“剧情人物”压过真实政治人物身份。

2. 周敦颐
   - 修正为：学术 / 历史典故 / 诗文出处 / 神交引用
   - 原因：《李语录》引用“出淤泥而不染”“莲，花之君子也”，属于明确文本引用，不应停留在“待复核”。

3. 韩孔厂
   - 修正为：诗文出处 / 神交引用
   - 原因：文中引用其“舌下无英雄，笔底无奇士”，属于纸上引用，不是直接关系。

4. 孔德成
   - 主类从“宗教”调整为“学术”
   - 保留：政治人物 / 宗教 / 诉讼司法 / 公共论战
   - 原因：《李语录》中是考试院长任命讽刺语境，主类用“宗教”容易误导。

本轮复核后保留不变
1. 马占山：军事人物 / 政治人物。此处与肯尼迪同作“被逼成英雄”的历史例子，不归入情治军警。
2. 萧伯纳：神交引用。符合此前专项抽查经验。
3. 林希翎：政治异议 / 朋友 / 见面。此处为政治评论与引语，暂不新增关系边。
4. 牟宗三：学术 / 公共论战 / 神交引用。此处为批评“玄学怪人”的论战语境。
5. 张良、黄石公：历史典故 / 神交引用保留；二者在本书中是典故比喻，不新增人物关系边。

校对经验
1. 真实历史人物如果也在小说剧本中出现，主类仍应优先保留其真实身份，小说身份放在后面。
2. “某某说”不一定是直接关系；如果是名言、古文、典故，应进入“诗文出处”或“神交引用”。
3. 政治讽刺语境中的人物，优先判断其现实身份和公共论战关系，不要被单一句子里的词带偏。
4. “宗教”类应谨慎做主类；孔德成这类人物更适合以学术、政治或制度身份为入口。
