本轮修正

1. 清理假人名“彭交情”。
   原文是“国民党认为我与彭交情很深”，这里的“彭”指彭明敏，“交情”不是名的一部分。已加入停用假名，不再进入人物图。

2. 修正林水泉身份。
   林水泉出现在谢聪敏回忆的审讯段落中，线索是“林水泉坐在牢里”，不是情治人员。已从“情治警务”改为“政治异议者 / 同案狱友 / 政治人物”。

3. 补入小林正成。
   小林正成是“住他押房隔壁”的日本囚犯，在谢聪敏由狱中带信、形成海外救援线索中有具体作用。补入“同案狱友 / 证人见证 / 人权救援”。

4. 校正军法人员链。
   聂开国原被误判为“律师代理”，本轮改为“司法人员 / 军事人物 / 诉讼”。张玉芳、王云涛、韩延年也统一校正为“司法人员 / 军事人物 / 诉讼”。其中韩延年此前未进入人物表，本轮补入。

5. 补细张俊宏身份。
   张俊宏在谢聪敏返台欢迎场景中出现，属美丽岛系、民进党政治圈人物。已定为“政治人物 / 政治异议者 / 媒体 / 公共论战”。

6. 保留既有判断。
   谢聪敏、魏廷朝、刘辰旦、李政一、郭荣文、吴忠信、詹重雄继续归入同案狱友与政治异议链；刘昭祥、魏以之保留情治警务；江鹏坚、康宁祥保持党外政治人物/公共论战相关分类。

本轮校对经验

1. “与某交情很深”这类句子容易把“某交情”误切成人名，后续看到“某关系”“某交往”“某案情”等结构也要警惕。
2. 同一句里有“特务说”“调查局”“警总”，不代表所有名字都是情治人员；要看该人是说话者、执行者，还是被审讯者提到的对象。
3. 军法官、军事检察官这种人物应同时保留“司法人员”和“军事人物”，否则会丢掉白色恐怖案件的制度语境。
4. 狱中传信、带信、海外刊登等人物，不一定是同案被告，但在人物关系图里可归入“同案狱友 / 证人见证 / 人权救援”一类，便于后续人工校对。
