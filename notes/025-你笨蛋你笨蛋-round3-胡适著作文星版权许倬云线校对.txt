身份边校正
1. 许倬云：学术 / 诉讼 / 公共论战。
   理由：本书核心语境是李敖具状控告许倬云造谣诽谤，不能只保留学术与论战身份。

2. 江冬秀：姻亲 / 出版 / 公共论战。
   理由：她作为胡适夫人出现，同时在胡适著作权益、出版争议与公开控诉链条中反复出现。

3. 胡颂平：学术 / 学术行政 / 出版 / 公共论战。
   理由：本轮语境是胡适纪念馆委员会、胡适著作权益与出版争议；先前落入“监狱行政”是旧语境词误导。

4. 魏岩涛：学术 / 学术行政 / 公共论战。
   理由：他作为胡适纪念馆委员会委员出现，不应因附近有“胡夫人”而误归姻亲。

5. 李筱峰：学术 / 公共论战。
   理由：文本中以“筱峰”反复出现，并与《胡适语粹》出版动机质疑相关，已补为独立人物并加别名。

6. 章学诚：学术 / 神交引用。
   理由：“章实斋先生年谱”中的章实斋应归并为章学诚，作为胡适著作目录中的被引用历史学者，不保留重复节点。

别名与合并
1. 胡夫人、胡适夫人、胡江冬秀 -> 江冬秀。
2. 筱峰 -> 李筱峰。
3. 章实斋 -> 章学诚。

误切清理
1. “都会发”来自“都会发现”，不是人物。
2. “时还要”来自“转手时还要”，不是人物。
3. 两者已加入停用名单，并在重跑后从数据和CSV中消失。

本轮校对经验
1. “夫人”附近不一定构成姻亲身份；如果人名处在委员会、校友会、著作权益会议名单中，应优先判断为学术行政、出版或公共论战线。
2. “秘书、所长、主任”一类身份词会继承旧语境，必须结合章节主线复核，不能把所有“所长”推向监狱行政或情治线。
3. 书名、年谱、索引中的号称名容易形成重复人物，例如“章实斋”；后续遇到字号、笔名、别称，优先归并到常用本名。
4. 许倬云这类“被控告者”即使主要社会身份是学者，也要加诉讼边，否则关系图看不出事件性质。
5. 出版版权线要单独看“著作、版权、付印、重版、委由、契约、公证”等词，不要只靠人物本业分类。
